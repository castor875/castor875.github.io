Insertar registros

Para insertar regisros en la base de datos debemos crear la consulta SQL que se encargara de guardar el registro

 
<?php

$conexion = mysql_connect("localhost","user","pass");
mysql_select_db("escuela",$conexion);
$sql="INSERT INTO alumnos(nombre,apellido,edad) VALUES('Sonia','Romo',20)";
mysql_query($sql);

>?

INSERT INTO:Se especifica que se ingresara un registro en en la tabla alumnos

VALUES:Especifica los valores de una o más filas de una tabla

mysql_query:Se utiliza para ejecutar consulta en la base de datos.
