Conexión con la base de datos

A continuación te explicamos con un ejemplo como realizar la conexión de una base de datos con php.

 
<?php

$link =mysql_connect("localhost","usuario","clave");
mysql_select_db("nombrebd",$link) OR DIE ("Error: No es posible establecer la conexión");

>?

Localhost: Se debe ingresar la dirección del servidor donde esta alojada la base de datos (web, dirección ip, localhost, etc).

Usuario: Se debe ingresar el usuario de acceso a la base de datos.

Clave: La clave correspondiente del usuario.

Nombrebd: Se debe escribir el nombre de la base de datos.
