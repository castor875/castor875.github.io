Eliminar registros

La instrucción DELETE se utiliza para eliminar registros existentes en una tabla

 
<?php

mysql_select_db("escuela",$conexion);
$sql="DELETE FROM alumnos WHERE id = '$id'";
mysql_query($sql);

>?

DELETE FROM: Especifica la tabla donde se encuentra el registro que se debe actualizar

WHERE: Condicion que se especifica del registro a eliminar.
